# 🤖 AI Website Companion

A premium RAG-powered chatbot that can "learn" from websites and answer questions based on their content.

## Features
- **RAG (Retrieval-Augmented Generation)**: Uses Google Gemini and ChromaDB to extract knowledge from URLs.
- **FastAPI Backend**: Robust API for authentication, scraping, and vector search.
- **Premium Gradio UI**: A high-end, modern interface with dark mode and glassmorphism.
- **Secure Auth**: Custom user registration and login integrated with the backend.

## How to use
1. Add your **Google Gemini API Key** in the **Settings** tab.
2. Login or Signup.
3. Go to **Manage Knowledge** and provide a URL to index.
4. Go to **Ask AI** and start chatting!
